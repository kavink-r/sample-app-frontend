This is a Demo application to practice CI/CD using Jenkins. This web application consists of a RestAPI, Front-end User Interface and MySQL database developed using Java SpringBoot, Typescript Angular and MySQL database instance is created in AWS RDS MySQL. The application is containerized using Docker and pushed to DockerHub. The initial step consists of creating an MySQL database instance in AWS RDS. After successfull creation of database instance, RestAPI is developed using Java SpringBoot. After successfull development of RestAPI, UserInterface is developed using Typescript Angular. Dockerfiles are created for both the application in order to proceed to build and deployment process. The entire build and deployment process is automated using Jenkins CI/CD tool by pulling the source code from GitHub, Building the application using Maven for RestAPi and NodeJS for User Interface, Creating docker images using Docker and pushing to DockerHub. An AWS cloud instance is created to host this Demo web application. The instance is installed with Docker engine and running. The docker images are pulled from DockerHub and executed in the docker engine of the cloud instance.
